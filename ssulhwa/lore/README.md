# SSULHWA Story Bible (v0.1)

이 문서는 기존 텍스트(역극/예전 것) 기반의 캐릭터/세계관/시나리오 초안을 정리한 작업본이다.
자동 생성 워크플로우를 만들 때 이 문서를 기준으로 데이터를 구조화할 수 있다.

## 범위
- 사일라즈/코오리 중심의 캐릭터 설정과 세계관을 우선 정리한다.
- 미가링/네로 등은 친구의 캐릭터이며, 크로스오버 범주로만 표기한다.
- 크로스오버 영역의 설정은 원저작자(친구)의 기준을 존중하며, 임의 변경하지 않는다.

## 태그 범례
- [canon] 기존 텍스트에서 확인된 내용
- [local] 사일라즈/코오리 중심 세계관에서 확정된 내용
- [xover] 친구 세계관에서 가져온 내용(크로스오버)
- [draft] 정합성을 위한 보강 추정
- [tbd] 결정 필요 또는 추가 확인 필요

## 구성
- `lore/world.md` 세계관/규칙/지명/세력/유물
- `lore/characters.md` 주요 인물 프로필 + 서브 캐릭터
- `lore/relationships.md` 관계도/갈등 구조
- `lore/scenario.md` 전체 시나리오 아크 구조
- `lore/templates.md` 작성 양식(캐릭터/세계관/사건/시나리오)
- `lore/visual/turnaround_specs.md` 캐릭터 삼면도(설명형)
- `lore/visual/illustration_briefs.md` 일러스트 및 씬 브리프
