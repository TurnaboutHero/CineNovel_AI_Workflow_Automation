# Phase 0-6: 이미지 배치·연출 규칙

## 목적
- 웹소설 본문에 삽입되는 이미지의 **아트 스타일, 배치 규칙, 연출 문법**을 정의한다.
- Phase 03(이미지 프롬프트 생성)과 Phase 04(이미지 생성)의 기준 문서 역할.
- style-guide.md의 `[IMG:유형-설명]` 마커와 illustration_briefs.md의 씬 목록을 실제 이미지로 연결하는 브릿지.

## 1. 아트 스타일 가이드

### 기본 화풍 [confirmed]
- **스타일**: 한일 하이브리드 일러스트 (한국 웹소설 삽화 + 일본 라노벨 일러스트의 중간)
- **선화**: 깔끔한 외곽선 + 내부 선 최소화. 만화적 과장보다 사실적 비례 우선
- **채색**: 풀채색 (풀 컬러). 그라데이션과 빛 표현 풍부. 셀 쉐이딩보다 소프트 쉐이딩 지향
- **질감**: 의상 질감(천, 금속, 모피) 표현 중시. 피부는 부드럽게, 무기/갑옷은 날카롭게
- **배경**: 씬 일러스트는 배경 필수. 캐릭터 단독 포트레이트는 단색/그라데이션 배경 허용

### 화풍 레퍼런스 키워드
```
korean webtoon illustration, light novel cover art,
detailed fantasy costume, soft shading, vibrant colors,
clean lineart, semi-realistic proportions,
atmospheric lighting, magical particle effects
```

### 지양 화풍
- 과도한 SD/치비 비례 (본편 삽화에서)
- 3D 렌더링 느낌 (과도한 광택/플라스틱 질감)
- 흑백 선화만 (커버/삽화 모두 풀채색)
- 서양 코믹스 스타일 (그림자 강조, 크로스해칭)

## 3. 이미지 삽입 규칙

### 삽입 기준 (style-guide.md 연동)
- **권당 3~5개** 이미지 삽입
- 과잉 삽입 지양 — 이미지는 "강조"이지 "설명"이 아님

### 삽입 타이밍

| 우선순위 | 삽입 시점 | 예시 |
|---------|----------|------|
| **필수** | 새 장소 첫 등장 | 아스트라 첫 도착 (7권) |
| **필수** | 감정 클라이맥스 | 자기 용서 순간 (23권) |
| **필수** | 새 캐릭터/형태 첫 등장 | 카르마 첫 등장 (6권), 하프 폴리모프 |
| **권장** | 전투 결정적 순간 | 마스터 웨폰 해방 (24권) |
| **권장** | 중요 아이템 획득/변화 | 마스터 웨폰 첫 접촉 (13권) |
| **선택** | 분위기 전환 | 일상→긴박, 밤→낮 전환 |
| **금지** | 이미 묘사로 충분한 장면 | 단순 대화, 이동, 설명 씬 |

### 배치 위치
- 해당 묘사 문단 **직후**에 배치 (묘사 → 이미지 → 반응/전개)
- 절대 문단 **중간**에 삽입하지 않음
- 이미지 전후로 `---` 구분선 사용

### 마커 형식 (style-guide.md 규칙 재확인)
```markdown
[본문 묘사 문단]

---
[IMG:유형-간결한_설명]
---

[후속 문단]
```

유형: `landscape` / `character` / `action` / `emotion` / `item`

### 조명 연출

| 상황 | 조명 방향 | 색온도 | 그림자 |
|------|----------|--------|--------|
| **일상/학원** | 자연광 (측면/상단) | 따뜻한 주광 | 부드러운 그림자 |
| **아스트라 야간** | 별빛 (상단) + 마나 결정 (하단) | 차가운 청+따뜻한 금 혼합 | 다방향 약한 그림자 |
| **전투** | 에너지 발광 (캐릭터 기준) | 에너지 색상에 따라 변동 | 강한 콘트라스트 |
| **가스라이팅/트라우마** | 상단 역광 | 차가운 청백색 | 얼굴에 강한 그림자 |
| **각성/해방** | 캐릭터 중심 발산 | 따뜻한 금+별빛 | 그림자 소멸 (빛이 모든 방향) |
| **회상** | 확산광 (방향 없음) | 세피아/따뜻한 톤 | 그림자 연함 |

### 반복 연출 패턴 (시각적 모티프)

| 모티프 | 의미 | 사용 씬 |
|--------|------|---------|
| **손등 문양 클로즈업** | 각성/성장/결의 | 서클업, 무기 해방, 결단 순간 |
| **하트 브로치 발광** | 어머니의 유산, 보호, 사랑 | 위기 극복, 소피아 회상 연결 |
| **별이 쏟아지는 하늘** | 우주 기운, 에스텔리아(별), 운명 | 아스트라 장면, 각성 장면 |
| **얼음 결정 클로즈업** | 코오리의 존재, 변화, 깨어남 | 시즌 2 티저, 코오리 관련 복선 |
| **검과 빛** | 마일스=검, 소피아=빛, 사일라즈=둘의 합 | 부자 장면, 가족 테마 |
| **균열/찢어짐** | 폭주, 카르마의 한계, 위기 | 폭주 씬, 카르마 폭주 |
| **나선형 패턴** | 또아리 조율, 에너지 통합, 성장 | 마법 발동, 서클업, 통합 씬 |

## 5. 씬 유형별 연출 가이드

### landscape (풍경)
- **비율**: 가로형 (16:9 또는 2:1). 스케일감 최우선
- **인물**: 없거나 실루엣/작은 크기로만. 풍경이 주역
- **용도**: 새 장소 등장, 분위기 전환, 세계관 소개
- **주의**: 판타지 세계관에 맞지 않는 현대적 요소 배제

### character (인물)
- **비율**: 세로형 (3:4) 또는 정방형 (1:1)
- **인물**: 1~2명 중심. 표정/포즈로 성격·관계 전달
- **배경**: 간소화 허용. 인물에 포커스
- **용도**: 첫 등장, 관계 변화, 성장 전후 대비

### action (전투/액션)
- **비율**: 가로형 선호, 세로도 가능 (수직 움직임 강조 시)
- **인물**: 동적 포즈. 모션 블러/잔상 효과 활용
- **에너지**: 에너지 시각화가 핵심. 에너지 컬러 체계 엄수
- **배경**: 전투 환경을 보여주되, 에너지 효과가 화면 지배
- **용도**: 마법 발동, 서클업, 무기 해방, 전투 결정타

### emotion (감정 하이라이트)
- **비율**: 자유 (씬에 맞게)
- **인물**: 클로즈업~미디엄샷. 표정/시선이 핵심
- **배경**: 최소화 또는 추상적 (감정에 맞는 색감/빛으로 대체)
- **조명**: 감정에 따라 극적 변화 (밝음=희망, 역광=고립, 따뜻함=화해)
- **용도**: 트라우마, 자기 용서, 부자 화해, 폭주 공포

### item (아이템/무기)
- **비율**: 정방형 또는 세로형
- **구도**: 아이템 중앙, 주변에 미세한 빛/입자 효과
- **디테일**: 룬 문양, 소재 질감, 발광 상태를 정밀 묘사
- **배경**: 단색 또는 그라데이션. 아이템에 집중
- **용도**: 마스터 웨폰, 하트 브로치, 억제 반지, 소피아의 문서

## 6. 웹소설 레이아웃 규칙

### 이미지 크기/해상도
- **권장 해상도**: 최소 1024×1024px (AI 생성 기준). 최종 출력은 플랫폼 규격에 맞춤
- **삽화 크기**: 본문 폭의 80~100%. 좌우 여백 유지
- **풍경 삽화**: 본문 폭 100% 사용 (풀 와이드)
- **인물/아이템**: 본문 폭의 60~80% (중앙 정렬)

### 이미지-텍스트 밸런스
- **권당 기준**: 약 10,000자 중 이미지 3~5장
  - 프롤로그/클라이맥스 권: 5장 (시각적 임팩트 극대화)
  - 내면/대화 중심 권: 3장 (텍스트 몰입 우선)
- **연속 삽화 금지**: 이미지 사이에 최소 800자 이상의 본문
- **첫 이미지 위치**: 권의 첫 500자 이내에 첫 이미지 (독자 시선 잡기)
- **마지막 이미지**: 클라이맥스 또는 엔딩 훅 직전 (여운 생성)

### 표지 규격
- **시즌 1 메인 표지**: 사일라즈 전신 포트레이트. 붉은 정장+마스터 웨폰+가문 문양 발광
- **권별 표지**: 해당 권의 핵심 씬 기반. 인물 중심 구도
- **표지 텍스트 영역**: 상단 1/4에 제목, 하단 1/8에 작가명/권수. 인물은 중앙~하단 배치

## 7. AI 이미지 생성 파이프라인 (architecture-v3_1.html Phase 03-04 연동)

### 생성 워크플로우
```
Phase 03: 이미지 프롬프트 생성 (Gemini)
├── illustration_briefs.md의 씬 목록 참조
├── turnaround_specs.md의 캐릭터 스펙 참조
├── 이 문서(image-direction.md)의 연출 규칙 참조
└── 출력: 씬별 상세 프롬프트 (영문)

Phase 04: 이미지 생성
├── 모델: architecture-v3_1.html 지정 모델 사용
├── 입력: Phase 03 프롬프트 + 네거티브 키워드
├── 일관성: 캐릭터 시드/LoRA 또는 레퍼런스 이미지 사용
└── 출력: 권별 3~5장 이미지 세트
```

### 프롬프트 변환 규칙
- illustration_briefs.md의 한국어 설명 → Phase 03에서 영문 상세 프롬프트로 변환
- **필수 포함 요소**: 캐릭터 외형 (turnaround_specs), 에너지 시각화 (이 문서), 장소 톤 (이 문서)
- **프롬프트 길이**: 100~200 토큰 권장. 핵심 요소 앞배치
- **일관성 키워드**: 캐릭터별 고정 외형 키워드를 모든 프롬프트에 삽입
