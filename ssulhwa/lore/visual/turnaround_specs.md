# Phase 0-3: 캐릭터 삼면도(설명형) 스펙

## 범위
- [local] 사일라즈/코오리 + 주요 조연 (마일스/카르마/소피아) 중심으로 제작한다.
- [xover] 친구 캐릭터는 필요 시에만 참고용으로 기록한다.

## 사일라즈 엔타리스

### 기본 정보
- 나이/인상: 27세 전후, 성숙하고 단정한 인상
- 체형: 슬림하고 길게 뻗은 체형. 어깨는 넓지 않으나 자세가 반듯함
- 전체 느낌: "귀족적 절제미" — 화려하지 않지만 품격이 있는 실루엣

### Front View
- **헤어**: 연갈색 계열, 중단발~단발. 단정하게 넘김. 이마 일부 드러남
- **눈**: 푸른 눈 (사파이어 톤). 차분하고 깊은 인상. 능력 발동 시 별빛처럼 발광
- **상의**: 붉은 정장 재킷 + 흰 셔츠 (대표 실루엣)
  - 재킷: 더블 브레스트 계열, 금 단추. 라펠이 날카로움
  - 셔츠: 흰색, 높은 카라. 넥타이 없음 — 정중하되 약간의 자유로움
- **하트 브로치**: 왼쪽 가슴 라펠에 착용. 금 테두리 + 붉은 보석 (마나 결정 내장). 능력 발동 시 은은한 붉은 빛
  - 크기: 엄지손톱 정도. 눈에 잘 띄지만 과하지 않은 크기
  - 디자인: 하트 외곽 + 내부에 미세한 룬 문양 (소피아의 손길 암시)
- **가문 문양 (손등)**: 왼손 등에 위치. 평소에는 상처/흉터처럼 보이는 은색 문양
  - 형태: 별을 중심으로 뻗는 기하학적 선 — 에스텔리아(별)+엔타리스 상징
  - 비발동: 은색/회색 흉터 질감
  - 발동: 금빛~별빛 발광 (강도에 따라 밝기 변화)
  - 각성/무기 해방: 강렬한 금빛+우주적 파란빛 혼합
- **장갑**: 미착용 (가문 문양 노출을 위해). 단, 정체 은닉 시 장갑 착용 변형 있음
- **하의**: 검은색 슬랙스, 깔끔한 실루엣
- **부츠**: 검은색 앵클부츠, 각진 디자인. 귀족적이면서 실용적

### Side View
- 슬림하되 허리가 잘록한 실루엣 (재킷이 강조)
- 자세: 등이 곧고 턱이 약간 내려감 — "듣는 자세"의 인상
- 재킷 뒤판: 허리선에서 약간의 절개 (테일 코트의 암시, 그러나 과하지 않음)

### Back View
- 재킷 등판: 중앙에 세로 절개선. 가문 문양은 등에 없음 (문양은 손등에만)
- 헤어 뒷모습: 뒷목이 깔끔하게 정리됨
- 전체: 뒤에서 보았을 때 "귀족 학생" 인상

### 복장 변형 (씬별)

| 변형 | 상황 | 주요 변화 |
|------|------|----------|
| **녹색 정장** | 일상/비공식 | 재킷 색상이 짙은 녹색으로 변경. 링크 오마주 |
| **초능력원 교복** | 학원 아크 | 군청색 기반 교복+망토. 하트 브로치는 유지 |
| **수련복** | 아스트라 훈련 | 흰 셔츠+검은 조끼, 동작 편의. 소매 걷어 가문 문양 노출 |
| **전투 장비** | 전투씬 | 붉은 정장 + 마스터 웨폰 장착 상태. 가문 문양 발광 |
| **법조인 (현대)** | 현대 설정 | 남색 3피스 정장 + 하트 브로치. 가문 문양은 장갑으로 은닉 |

### 마스터 웨폰 디자인
- **기본 형태**: 한손 장검 (바스타드 소드). 날 길이 약 80cm
- **색상**: 은백색 금속 + 손잡이에 금/청 장식. 가드 부분에 미세 룬 문양
- **전설 룬**: 날 표면에 평소 보이지 않으나, 발동 시 별빛 색상의 룬 문양이 떠오름
- **형태 전환**: 검→방패→활→총 등. 전환 시 금빛 입자로 분해 → 재조립 시각 효과
- **해방 상태**: 날에서 우주적 빛 (별빛+성운 색상)이 흘러넘침. 룬이 모두 활성화

### 에너지 시각 효과
- **마나 사용**: 손끝/룬 주변에 속성별 색상 빛 (화염=적, 냉기=청 등)
- **우주 기운 사용**: 가문 문양 발광 + 주변 공간에 별빛/성운 같은 입자. 아지랑이 효과
- **기 사용**: 신체 주변에 투명~반투명 오러. 바람 같은 기류 흐름
- **3에너지 통합**: 세 효과가 겹침 — 손끝의 마나 빛 + 문양의 별빛 + 신체의 기류가 하나로 수렴

## 츠메타이 코오리

### 기본 정보
- 나이/인상: 중/고등학생 정도 (14~16세 인상). 순하고 밝은 첫인상
- 체형: 슬림, 소년~청년. 사일라즈보다 머리 하나 작음
- 전체 느낌: "겨울 왕자" — 하얗고 차가운 색감이지만 표정은 따뜻함

### Front View
- **헤어**: 백발 (순백~은발). 약간 길어 귀 아래까지. 자연스럽게 흐트러진 스타일
- **눈**: 붉은 눈 (루비 톤). 동공이 세로로 약간 길음 (드래고니안 특징, 미묘하게)
- **귀**: 뾰족한 귀 — 드래고니안 특유. 엘프와 유사하여 혼동 유발
  - 엘프 귀보다 약간 짧고 두꺼움 (미세한 차이. 일반인은 구분 못함)
  - 귀끝에 미세한 비늘 질감 (매우 가까이 봐야 보임)
- **상의**: 두터운 코트 (겨울 의상 느낌)
  - 카라: 큰 퍼 카라 (백색~은색 모피). 얼굴을 반쯤 감쌈
  - 코트 색상: 짙은 남색~흑색. 안감은 옅은 하늘색
  - 장식: 은색 체인 + 별/눈결정 형태 브로치 (가슴 중앙)
- **억제 반지**: 오른손 검지에 착용
  - 디자인: 은빛 금속 밴드 + 드워프 공학 룬이 정밀하게 새겨짐 (기하학적 선 패턴)
  - 보석: 작은 사파이어색 마나 결정 1개 삽입
  - 기능 표현: 평소 은은한 푸른 빛 맥동 (마나 흡수 중)
  - 한계 시: 룬이 붉게 변하며 균열 표현
- **드래곤 하트 장신구**: 목걸이 형태 (쇄골 위치)
  - 디자인: 은색 용 발톱이 청색 보석을 감싼 펜던트
  - 보석: 얼음처럼 투명한 청색. 내부에 서리 같은 무늬
  - 감정 격동 시: 보석 내부가 요동치며 빛남
- **하의**: 검은 바지, 부츠

### Side View
- 슬림하고 가벼운 실루엣. 코트가 체형을 감싸 작아 보임
- 퍼 카라가 목/어깨를 두르는 볼륨감
- 뾰족한 귀가 머리카락 사이로 보임

### Back View
- 코트 등판: 눈/별 문양이 자수로 새겨짐 (사일라즈가 준 코트라면 엔타리스 문양도 가능)
- 헤어: 뒷모습에서 백발이 특히 눈에 띔
- 꼬리/날개: **평상시 없음** (억제 반지에 의해 봉인)

### 복장 변형 (씬별)

| 변형 | 상황 | 주요 변화 |
|------|------|----------|
| **일상복** | 학원/도시 | 코트 대신 후드 달린 재킷. 귀를 후드로 가림 (정체 은닉) |
| **수련복** | 훈련 시 | 가벼운 긴팔 셔츠+조끼. 동작 편의. 억제 반지는 항상 착용 |
| **하프 폴리모프** | 전투/본능 발동 | 이마에 은백색 뿔 1쌍 + 등에서 얼음 결정 날개 + 손등/팔에 은색 비늘 |
| **풀 폴리모프** | 4단계 동기화 | 동양 용 계열 — 긴 몸체, 은백색 비늘, 얼음빛 갈기, 붉은 눈 유지 |

### 하프 폴리모프 상세
- **뿔**: 이마 양쪽에서 뒤로 휘어지는 은백색 뿔. 얼음 결정 질감
- **날개**: 등에서 돋아나는 반투명 얼음 결정 날개 (나비보다는 드래곤 형태). 빛에 따라 무지갯빛 반사
- **비늘**: 손등/팔뚝/목/볼에 은색~청백색 비늘이 부분적으로 돋음
- **발톱**: 손톱이 날카로운 발톱으로 변화
- **눈**: 붉은 눈이 더 강렬해지며, 동공이 완전한 세로 슬릿으로 변화
- **냉기**: 호흡이 하얀 입김으로 보이고, 주변 온도 하강 묘사

### 에너지 시각 효과
- **아이스 브레스**: 입에서 뿜어내는 극저온 냉기. 푸른빛~백색 결정 입자
- **얼음 장막**: 주변에 얼음 결정이 공중 부유하며 방어막 형성
- **얼음 메테오**: 상공에서 거대 얼음 결정이 형성되어 낙하
- **드래곤 하트 공명**: 펜던트가 강렬하게 빛나며, 눈/서리가 소용돌이치는 효과
- **억제 반지 한계**: 룬이 붉어지며, 반지 주변에 서리가 피어오르고, 코오리의 손에 비늘이 돋기 시작

## 마일스 엔타리스

### 기본 정보
- 나이/인상: 50대 초반이나 40대로 보임. 단련된 전사의 위엄
- 체형: 장신, 다부진 근육. 사일라즈보다 어깨가 넓고 존재감이 큼
- 전체 느낌: "살아있는 무기" — 서 있기만 해도 위압적, 그러나 냉정하지 잔인하진 않음

### Front View
- **헤어**: 짙은 갈색 (사일라즈보다 어두운 톤). 짧게 정리, 뒤로 넘김
- **눈**: 날카로운 회청색. 감정 표현이 적어 차갑게 보임
- **상의**: 군복 스타일 코트 (검은색 기반, 금 트리밍)
  - 에폴릿 (어깨 장식): 금색. 개국공신 지위 상징
  - 가슴에 훈장/메달류 (절제적으로 1~2개)
- **무기**: 등에 가로 착용 또는 허리에 찬 장검
  - 마일스의 검: 직선형 양날검. 장식 적음. 실전 지향 디자인
  - 검에서 기의 오러가 미세하게 흐르는 묘사 가능
- **가문 문양**: 오른손 등. 사일라즈와 같은 디자인이나 더 진하고 선명함
- **하의**: 검은 군복 바지 + 장화

### Side/Back View
- 넓은 어깨와 곧은 등이 강조되는 실루엣
- 코트 뒤판에 엔타리스 가문 문양 자수 (공식 예복 시)
- 검을 등에 맸을 때 X자 벨트 디자인

### 에너지 시각 효과
- **기 사용**: 검날을 따라 투명~은색 기류가 흐름. 베기 시 기의 궤적이 잔상으로 남음
- **우주 기운 사용**: 가문 문양 발광 + 눈 주변에 미세한 별빛. 마일스는 기로 안정화하므로 사일라즈보다 절제됨
- **전력 발휘**: 검기(劍氣)가 반달 형태로 날아감. 은색~금색 빛

## 카르마 엔타리스

### 기본 정보
- 나이/인상: 50대 중반. 마일스보다 2~3살 위. 외견상 점잖은 귀족
- 체형: 마일스보다 약간 작고 마른 편. 그러나 나름의 위엄
- 전체 느낌: "광택 나는 칼집" — 겉은 번듯하나 안에 날이 숨어있음

### Front View
- **헤어**: 짙은 갈색 (마일스와 유사), 약간 길어 귀를 덮음. 올백으로 정리
- **눈**: 회색 톤. 마일스보다 따뜻해 보이지만, 자세히 보면 계산적
- **상의**: 격식 높은 예복 스타일. 검은색 + 자주색(보라) 트리밍
  - 마일스의 "군인" 느낌과 대비되는 "귀족/정치인" 느낌
  - 카라가 높고, 단추를 끝까지 잠금 — 자기 통제/억압의 시각적 표현
- **가문 문양**: 왼손 등. 같은 엔타리스 문양이나, 억압의 결과로 색이 약간 바래 보임
- **장갑**: 자주 착용 — 문양을 의도적으로 가리는 습관 (열등감의 표현)

### 시각적 대비 (마일스 vs 카르마)
- 마일스: 검은 군복 + 금. 직선적. 검 착용 → **전사**
- 카르마: 검은 예복 + 자주. 곡선적. 지팡이 또는 빈손 → **정치인**

### 에너지 시각 효과
- **생 초능력 사용**: 가문 문양이 불안정하게 명멸. 빛이 고르지 않음 (매개체 없음의 시각 표현)
- **억압 상태**: 손/팔 주변에 균열 같은 빛이 새어나옴 — "터지기 직전"의 느낌
- **폭주 시**: 별빛이 아니라 **검은 자주빛+금빛이 뒤섞인 불안정한 빛**. 공간 왜곡 효과

## 소피아 (회상/유품 등장용)

### 기본 정보
- 나이/인상: 20대 후반~30대 초반 (사망 시). 온화하고 총명한 인상
- 체형: 보통. 너무 강하지도, 약하지도 않은 균형 잡힌 체형
- 전체 느낌: "따뜻한 지성" — 학자이자 어머니이자 외교관

### Front View
- **헤어**: 연갈색 (사일라즈와 같은 톤). 긴 머리를 부드럽게 묶음 (하프업)
- **눈**: 따뜻한 갈색~호박색. 사일라즈의 푸른 눈은 아버지 쪽 유전
- **상의**: 드레스 + 숄 스타일. 아이보리~연한 금색 톤
  - 관료/학자풍: 소매에 미세한 룬 자수 (지식인의 표현)
  - 가슴에 소규모 브로치 (사일라즈의 하트 브로치와 쌍인 디자인 가능)
