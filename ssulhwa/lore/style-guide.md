# Phase 0-5: 문체 · 출력 형식 가이드

## 시점/POV 구조 [confirmed]
- **전지적 작가 시점 + 시즌별 주인공 포커스**
- 기본: 전지적 시점 (모든 캐릭터 내면 접근 가능)
- S1: 사일라즈에게 카메라가 더 가까이 → 내면/감정 더 깊게
- S2: 코오리에게 카메라 이동 → 코오리 내면 더 깊게
- 다른 캐릭터의 내면도 접근 가능하되, 주인공보다 빈도/깊이를 조절

## 문체 톤 [confirmed]
- **서정적/상세 묘사 기반 + 상황별 완급 조절**
- 기본: 감정/분위기/공간 묘사 풍부. 독자가 장면을 "보는" 느낌
- 전투씬: 상세 기술 묘사 (에너지 흐름, 신체 감각, 공간 왜곡까지 묘사)
- 내면씬: 독백·감정 흐름을 깊이 있게. 트라우마/자기 대면 장면은 특히 밀도 높게
- 일상씬: 호흡 가볍게, 캐릭터 간 관계성 드러내는 대화 중심
- 긴박한 장면: 짧은 문장, 빠른 호흡으로 템포 조절

## 문장 스타일 [confirmed]
- **기본 호흡**: 중간 길이 (만연체 X, 지나치게 짧은 호흡 X)
- **묘사 밀도**: 핵심 장면은 풍부하게, 이동/전환은 간결하게
- **비유/은유**: 세계관에 맞는 비유 사용 (별, 빛, 에너지 흐름, 시간 등)
- **금지**: 현대 인터넷 용어, 이모티콘, 메타 발언
- **지양**: 같은 표현 반복, 과도한 감탄사, 설명충(세계관 설정을 대사로 장황하게 풀기)

## 대사 스타일 [confirmed]

| 캐릭터 | 말투 | 특징 |
|--------|------|------|
| **사일라즈** | 격식 있는 존댓말 | 정중하고 차분. 감정이 격해져도 말투는 유지. 어머니(소피아)를 닮음 |
| **코오리** | 존댓말 + 순한 말투 | 예의 바르고 농담도 함. 드래곤 본능 발동 시 말투 변화 |
| **마일스** | 논리적·엄격한 어조 | 감정 표현 서툼. 짧고 단정적. 소피아 사후 더 딱딱해짐 |
| **카르마** | 격식 + 비아냥 | 겉으로는 점잖지만 말 속에 칼날. 가스라이팅 화법 ("너를 위해서 하는 말이다") |
| **소피아** (회상) | 부드러운 존댓말 | 따뜻하지만 핵심을 찌르는 말. 논리적이되 감정에 공감하는 화법 |

## 분량 기준 [confirmed]
- **1권당**: 약 10,000자
- **시즌 1**: 총 42권 (총 약 42만자)
- **권당 목표**: 핵심 사건 1개 + 감정 전개 1개 + 떡밥 1개

## 에너지/전투 묘사 가이드 [confirmed]
- **마나 사용**: 변환 과정을 묘사 ("마나가 열에너지로 바뀌며 손끝에서 불꽃이 피어올랐다")
- **우주 기운 사용**: 현실 간섭/왜곡을 묘사 ("공간이 울렁이며 중력의 방향이 비틀렸다", "별빛 같은 기운이 손등의 문양에서 퍼져나갔다")
- **기 사용**: 신체 감각을 묘사 ("호흡에 맞춰 근육 안을 흐르는 기류가 팔 끝까지 뻗었다")
- **3에너지 통합**: 세 감각이 겹치는 묘사 ("마나의 회로 위로 별빛이 흐르고, 호흡의 리듬이 그 모든 것을 하나로 엮었다")
- **서클 가속**: 심장 부담을 신체 감각으로 ("가슴 속 고리가 미친 듯이 회전하며 심장이 비명을 질렀다")
- **폭주**: 우주 기운 = 공간 왜곡 + 정신 혼란 묘사 ("세상이 찢어지는 듯한 감각. 자신의 의지가 아닌 무언가가 몸을 지배하려 했다")

## 출력 마크다운 형식

```markdown
# EP-XX: [에피소드 제목]

> [에피소드 한 줄 요약]

---

[본문 텍스트]

[IMG:landscape-아스트라_전경] ← 이미지 삽입 위치 마커

[본문 계속]

---

**다음 화 예고**: [한 줄 티저]
```

## 이미지 삽입 마커 규칙 [draft]
- 형식: `[IMG:유형-설명]`
- 유형: `landscape` (풍경), `character` (인물), `action` (전투/액션), `emotion` (감정 하이라이트), `item` (아이템/무기)
- 삽입 기준: 권당 3~5개 (씬 전환, 감정 클라이맥스, 새 장소/캐릭터 등장 시)
- 배치: 해당 묘사 문단 직후

## 지양 표현 리스트
- "갑자기" 남발 (전환은 묘사로 처리)
- "~했다" 반복 (문미 변화 필요)
- 세계관 설정을 대사로 직접 설명하기 (자연스러운 상황 속에서 보여주기)
- 현대 속어/인터넷 밈
- 캐릭터가 자기 능력을 스스로 설명하는 장면 (다른 캐릭터의 시선/반응으로 대체)
