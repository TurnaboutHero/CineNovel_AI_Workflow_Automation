# Phase 0-1: 장르 · 레퍼런스 설정

## 장르
- **메인 장르**: 판타지 성장물 (Fantasy Coming-of-Age)
- **서브 장르**: 가문/혈통물, 마법 아카데미, 가족 드라마
- **플랫폼**: 웹소설 (씬노벨/웹툰 확장 가능)

## 톤 & 무드
- **기본 톤**: 서정적·상세 묘사 기반. 감정/분위기/공간 묘사 풍부
- **전투**: 상세 기술 묘사 (에너지 흐름, 신체 감각, 공간 변화까지 그려줌)
- **내면**: 깊은 심리 묘사. 트라우마·자기 용서·가족 관계의 감정선 중시
- **무드 스펙트럼**:
  - 밝음 ←————————→ 어두움
  - 중간~약간 어두움 (성장통, 트라우마, 가스라이팅 포함)
  - 그러나 궁극적으로 **희망적** — "인간 찬가" 테마

## 장르 컨벤션 (따를 것 / 비틀 것)

**따를 것**:
- 명확한 성장 단계 (Stage 1→2→3→4)
- 스승-제자 관계를 통한 기술 전수
- 결정적 순간의 각성/서클업
- 라이벌/동료와의 비교를 통한 동기 부여
- 권선징악의 기본 구조 (인간 찬가)

**비틀 것**:
- "천재가 노력까지" 클리셰 → 사일라즈는 천재이되 **트라우마로 봉인된 천재** (심리적 장벽이 핵심)
- "악역 삼촌" 단순 악역 → 카르마는 **자기도 피해자**(열등감/폭주 공포), 용서 딜레마
- "아버지의 그림자" → 마일스는 악인이 아니라 **소통 불능의 좋은 아버지**, 화해가 목표
- "파워업이 곧 성장" → 서클업은 **기술적 성장**, 진짜 성장은 **내면 극복** (둘 다 필요)

## 타겟 독자
- **1차 타겟**: 10대 후반~30대, 판타지 웹소설 독자
- **키워드**: 성장물, 가문/혈통, 마법 체계, 가족 드라마, 내면 갈등
- **차별점**: 에너지 체계의 과학적 구조감 + 깊은 가족/심리 드라마의 결합
- **독자 경험 목표**: "사일라즈가 마스터 웨폰을 해방하는 순간, 울컥해야 한다"
